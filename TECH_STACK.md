# B 站视频总结器 - 技术栈文档 🛠️

## 项目概述

输入 B 站视频链接，自动提取字幕（或 Whisper 语音识别），使用 LLM 生成结构化学习笔记。

---

## 核心技术栈

### 1. 后端框架

| 工具 | 版本 | 作用 |
|------|------|------|
| **Python** | 3.12+ | 主要编程语言 |
| **Flask** | ≥2.3.0 | Web 服务器框架，提供 API 和静态文件服务 |
| **Flask-CORS** | ≥4.0.0 | 跨域资源共享支持 |

---

### 2. 字幕提取模块

| 工具 | 版本 | 作用 |
|------|------|------|
| **requests** | ≥2.31.0 | HTTP 客户端，调用 B 站 API |
| **yt-dlp** | ≥2024.0.0 | 视频/音频下载工具，用于无字幕时提取音频 |

**B 站 API 端点：**
- `https://api.bilibili.com/x/web-interface/view` - 获取视频信息（标题、CID）
- `https://api.bilibili.com/x/player/wbi/v2` - 获取字幕列表

---

### 3. 语音识别模块

| 工具 | 版本 | 作用 |
|------|------|------|
| **openai-whisper** | ≥20231117 | OpenAI 开源的语音识别模型，本地运行 |
| **torch** | 2.10.0 | PyTorch 深度学习框架，Whisper 的依赖 |
| **numpy** | ≥2.4.0 | 数值计算库 |
| **numba** | ≥0.64.0 | JIT 编译器，加速音频处理 |
| **ffmpeg** | (系统级) | 音频编解码工具，yt-dlp 依赖 |

**Whisper 模型选项：**
| 模型 | 速度 | 精度 | 适用场景 |
|------|------|------|----------|
| `tiny` | ⚡⚡⚡ 最快 | ⭐⭐ 一般 | 快速测试，短视频 |
| `base` | ⚡⚡ 快 | ⭐⭐⭐ 不错 | 日常使用推荐 |
| `small` | ⚡ 中等 | ⭐⭐⭐⭐ 好 | 追求精度 |
| `medium` | 🐌 较慢 | ⭐⭐⭐⭐⭐ 很高 | 重要内容 |
| `large` | 🐌🐌 最慢 | ⭐⭐⭐⭐⭐ 最高 | 专业需求 |

---

### 4. 内容总结模块

| 工具 | 版本 | 作用 |
|------|------|------|
| **openai** | ≥1.0.0 | OpenAI SDK（兼容模式），调用 LLM API |

**LLM 服务：**
- **阿里百炼 (DashScope)** - Qwen 系列模型
- API 兼容 OpenAI 格式
- 支持模型：`qwen-plus`, `qwen-turbo`, `qwen-max` 等

---

### 5. 前端技术

| 技术 | 作用 |
|------|------|
| **HTML5** | 页面结构 |
| **CSS3** | 样式设计（渐变背景、卡片布局、动画） |
| **JavaScript (原生)** | 交互逻辑，无需框架 |
| **Fetch API** | 异步 HTTP 请求 |

**前端功能：**
- 视频链接输入
- Whisper 开关和模型选择
- API Key 输入（可选）
- 实时状态显示
- Markdown 输出预览
- 一键复制/下载

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## API 接口

### `POST /api/summarize`

**请求体：**
```json
{
  "url": "https://www.bilibili.com/video/BV1xx411c7mD",
  "api_key": "sk-xxx",  // 可选
  "use_whisper": true,
  "whisper_model": "tiny"
}
```

**响应：**
```json
{
  "success": true,
  "title": "视频标题",
  "summary": "# 视频标题\n\n## 📌 核心要点...",
  "message": "✅ 成功提取 CC 字幕（120 条）"
}
```

## 性能参考

| 场景 | 耗时 |
|------|------|
| 有 CC 字幕 | 5-10 秒 |
| 无字幕 + Whisper Tiny | ~1 分钟/10 分钟视频 |
| 无字幕 + Whisper Base | ~3 分钟/10 分钟视频 |
| 无字幕 + Whisper Small | ~5 分钟/10 分钟视频 |

## 技术亮点

1. **自动降级策略**：优先 CC 字幕，无字幕自动切换 Whisper
2. **本地隐私保护**：Whisper 本地运行，音频不上传第三方
3. **临时文件清理**：自动清理下载的音频，不占用空间
4. **零框架前端**：原生 JS 实现，无构建步骤
5. **跨平台支持**：Python + 虚拟环境，macOS/Windows/Linux 通用
